Setup:
1. Copy the contents of the RentMyThing folder into the xampp\htdocs directory.
2. Open XAMPP Control Panel and start the Apache and MySQL modules.
3. Open phpMyAdmin and create a database named 'test'.
4. Import the contents of the database, choosing the file test.sql located in the root directory of the project as the source.
5. Open a browser to the url 'localhost'.
6. Login to an existing user account (email: serge@email.com , password: serge) or register as a new user.
